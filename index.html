SmartStock: Centralized Access to Your Home Inventory

SmartStock provides a seamless, secure platform to track, manage, and analyze every item in your home. Built with a focus on real-time data, predictive insights, and a robust PostgreSQL backend.

Core Functionalities & Technical Focus

Robust PostgreSQL Backend

  • Secure Storage: User and inventory data safely stored in PostgreSQL.
  • Argon2 Hashing: Passwords are encrypted using argon2 before saving.
  • Dynamic Data: Categories auto-update based on added inventory.
  • FastAPI API: Communication via REST endpoints for login & registration.

Interactive Dashboard Management

  • Real-time display of items in responsive layout.
  • Inline editing and auto updates from backend API.
  • Critical warnings for low stock and expiry alerts.

Insightful Analytics & Expiry Control

  • Visual analytics with Chart.js.
  • Highlight soon-expiring items.
  • Modern animated UI with consistent theming.
